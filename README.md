# Файловая система miniBtrfs
## Описание 
Проект miniBtrfs представляет собой пользовательскую файловую систему, функционирующую через интерфейс FUSE. Основой хранения данных и метаданных выступает структура B-дерева. Система поддерживает базовые файловые операции, включая создание иерархии директорий, чтение и запись файлов.
# Архитектура

+ Суперблок. Обеспечивает персистентное хранение состояния файловой системы и управление распределением свободного дискового пространства.
+ B-дерево. Осуществляет строгую типизированную индексацию и поиск файловых объектов.
+ Модуль FUSE. Транслирует вызовы виртуальной файловой системы операционной системы во внутренние команды ядра miniBtrfs.

# Использование

```bash
make a
```
Перед началом работы необходимо инициировать структуру файловой системы внутри файла образа. Генерация суперблока и корневого каталога выполняется специализированной утилитой `mkfs.minibtrfs`.
```bash
./mkfs.minibtrfs fs.img <размер файла>
```
Монтирование созданного образа в виртуальную файловую систему операционной системы производится путем запуска основного исполняемого файла с указанием пути к образу и целевой точки монтирования. Для обеспечения корректности транзакций B-дерева требуется использование флага однопоточного режима.
```bash
mkdir -p /tmp/fs/mnt
./btfs fs.img /tmp/fs/mnt -f
```
